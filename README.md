# ESP32-C6 Web Server PoC

This project is a Proof of Concept (PoC) for an ESP32-C6 web server discoverable over mDNS at `mm.local`.

## Overview

The goal of this project is to demonstrate the networking capabilities of the ESP32-C6 using the ESP-IDF framework. It implements a basic web server that connects to a Wi-Fi network and registers itself via mDNS for easy discovery on a local network.
